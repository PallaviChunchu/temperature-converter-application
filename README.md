# 🌡️ Temperature Converter Application

## Project Overview

Temperature Converter Application is a modern front-end web application that converts temperatures between Celsius (°C), Fahrenheit (°F), and Kelvin (K). The application provides instant and accurate temperature conversions with a clean, responsive, and interactive user interface.

The project demonstrates front-end development skills using HTML, CSS, and JavaScript, including DOM manipulation, event handling, responsive design, animations, and user-friendly UI components.

## Features

### Temperature Conversion
- Convert Celsius to Fahrenheit & Kelvin
- Convert Fahrenheit to Celsius & Kelvin
- Convert Kelvin to Celsius & Fahrenheit
- Instant and accurate calculations

### User Interface
- Modern glassmorphism design
- Attractive gradient background
- Responsive layout for desktop, tablet, and mobile
- Interactive input fields
- Font Awesome icons
- Hover animations and smooth transitions

### User Experience
- Enter key support for quick conversion
- Dynamic unit symbol update
- Input validation
- Default conversion values displayed
- Clean and easy-to-use interface

## Installation

Clone the repository:

```bash
git clone https://github.com/PallaviChunchu/temperature-converter-application.git
```

Navigate to the project folder:

```bash
cd temperature-converter-application
```

Open the project in your preferred code editor.

Run the project by opening the `index.html` file in your preferred web browser.
